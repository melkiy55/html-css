Tags úteis

A tag PRE por outro lado considera os espaços na hora de passar
para o site, você pode subinhar todo o texto e utilizando shift+
tab, você move todo o texto para trás, se usar só o tab vai para frente.

citações simples

A tag q também sublina, mas não é recomendado usala, ao invés de
usala, use blockquote.

Citações

A tag blockquote e usada para citações, ela poen aspas no texto
envelopado, além disso, ela também afasta um pouco o texte, se
usar cite você pode mandar a url, ela não aparece na página web, mas de forma intena sabemos que esta lá.

Abreviações

A tag abbr é usada para explicar siglas ou algo que o leito possa
ter duvidas, ela fica na página de forma não visivel, só quando não
se passa o mause em cima

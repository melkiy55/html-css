Listas e seus tipos

Lista ordenadas

Lista ordenada são aquelas organizadas, que a mudança de posição
faz diferença, exemplo;

  1. Acordar as 9;0 hr
  2. Levantar da cama
  3. Escovar os dentes
  4. Tomar café
  5. Pegar sol
  6. Ir na escola
  7. Arrumar a casa
  8. Estudar html
  9. estudar php
  10. Estudar Idiomas
  11. estudar para concurso

Esses são exemplos de uma lista ordenada, a vantagem de usar
o'li' é que se houver alterações na lista ou um erro nela,
o li muda toda ela.

Lista não ordenada

A tag usada para uma lista não ordenada é 'ul' esse tipo de lista
não precisa precisa seguir uma ordem, caso você esque-
algo não faz diferença por depois, exemplo:

  • maça
  • Abacate
  • Cevada
  • Grãos
  • Pepino
  • Arroz
  • Feijão
  • Banana
  • Biscoito
  • Bolo
  • Adoçante

Na 'ul' ou na 'ol' pode se usar a tag 'type', essa tag permite
que você mude a númeração, podendo usar número romanos
,letras, maíuscula ou miniscula.

Na lista 'ul' você não usa números, nem romanos, nem cardiais
, nesse tipo de lista se usa disco, circulo ou quadrado
disc, cicle ou square.
